## Git

1. Git使用  
   > `git status`  
   > `git add *`  
   > `git commit -m "msg"`  
   > `git push -u origin main`  

2. Git访问  
   > 获取global.ssl.fastly地址  
   > `http://github.global.ssl.fastly.net.ipaddress.com/#ipinfo`  
   > 获取Github地址  
   > `http://github.com.ipaddress.com/#ipinfo`  
   > 在Hosts文件中添加映射关系，重启  
   > `C:\Windows\System32\drivers\etc\hosts`
