# prototype-project

## Git

1. Git使用  
   > `git status`  
   > `git add *`  
   > `git commit -m "msg"`  
   > `git push -u origin main`  

2. Git访问  
   > 查询IP地址网站 `https://www.ipaddress.com/`  
   > `github.com` 和 `github.global.ssl.fastly.net` IP地址  
   > 在Hosts文件中添加IP映射关系  
   > `C:\Windows\System32\drivers\etc\hosts`  
   > 刷新DNS缓存  
   > `ipconfig /flushdns`
